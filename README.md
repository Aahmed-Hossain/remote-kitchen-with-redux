# Live Link: https://remote-kitchen.vercel.app

# To Run on the local m/c.
```bash
git clone https://github.com/Aahmed-Hossain/remote-kitchen-with-redux.git
cd remote-kitchen
npm install
npm run dev
```
## Tech Stack
-Next.js
-TypeScript
-Redux
-Redux Toolkit
-React-toastify
-Tailwindcss
-Material Ui
-React-hook-form
